# Deven's Job Board

Curated job listings for A-Level graduates in the UK — engineering, finance, consulting, marketing & operations.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repo → Deploy
4. Share the URL with Deven

## Local Development

```bash
npm install
npm run dev
```

## Updating Jobs

Edit `src/App.jsx` → update the `ALL_JOBS` array → commit & push → auto-deploys.
